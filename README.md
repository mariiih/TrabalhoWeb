O "Organizando a Vida" foi projetado com um visual estilo “Kanban minimalista” para facilitar a visualização das minhas tarefas pessoais e com um tema dark, que tem muito a ver com o meu estilo. 

Com esse sistema, consigo gerenciar minha rotina, controlar meus prazos e minha produtividade. É possível filtrar as tarefas por data e hora, prioridade, nome ou data de criação, além de listar somente o que está em atraso. O projeto também permite editar e excluir cada tarefa com ícones destacados. 

No topo da página, coloquei botões para facilitar ainda mais: o “Listar todas as tarefas” (que organiza por ordem de data e hora), o “Tarefas Urgentes” (que filtra só o que é prioridade) e o “Limpar tudo”, para deletar todas as tarefas de uma vez. 

A frase no final é uma referência à personagem Dory, do filme “Procurando Nemo”. No filme é algo que ela sempre repete quando está em alguma situação difícil. Mencionei na página com intuito motivacional, já que me identifico bastante com a personagem.  

Dessa forma, o “Organizando a Vida” foi desenvolvido inicialmente com objetivo acadêmico, mas se tornou parte do meu cotidiano, personalizado especialmente para me ajudar com a organização do meu dia a dia.
